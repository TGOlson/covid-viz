# covid-viz

http://tgolson.com/covid-viz

Dashboard for various covid visualizations. Data from by Johns Hopkins CSSE (https://github.com/CSSEGISandData/COVID-19)

### dev

Setup
```
$ npm run start
```

Deploy
```
$ npm run deploy
```
